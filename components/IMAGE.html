参数 类型 默认值 描述
layerId String tile-layer-${uuid()} 图层id
geoImage Boolean false 是否采用ol-ext的GeoImage (opens new window)图层。
source Object 若geoImage不设置或为false。source参数参考 ol/source/ImageStatic (opens new window)。若geoImage为 true时，source参数参考ol.source. GeoImage (opens new window)
